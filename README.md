# Body Fat Calculator with Image Recognition

A mobile application built with **Java** and **XML** that calculates body fat percentage, **BMI** (Body Mass Index), and **FFMI** (Fat-Free Mass Index). The app uses **image recognition** and features a **local database** to store and track user stats over time for fitness and health assessments.

## Features

- **Body Fat Percentage Calculation**: Estimate your body fat percentage based on physical data and image recognition.
- **BMI & FFMI Calculations**: Compute your **BMI** and **FFMI** to assess overall health and muscle mass.
- **Image Recognition**: Analyze images to extract data for accurate calculations of body fat and other metrics.
- **Local Database**: Stores user stats locally for tracking progress over time.
- **User Profiles**: Allows users to create profiles and store their measurements and stats.
- **User-Friendly Interface**: Designed with **XML** for a clean, intuitive user experience.
- **Fitness & Health Insights**: Helps users track and evaluate their fitness and health progress.

## Technologies Used

- **Java**: Programming language used for the app's core functionality.
- **XML**: Used for designing the user interface.
- **SQLite**: Local database used for storing user stats and data.
- **Image Recognition**: Leveraged for analyzing body images and assisting in calculating body fat percentage.
- **Android Studio**: IDE used for building and testing the mobile application.
